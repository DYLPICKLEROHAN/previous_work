Summary¶

Problem defenition

The problem was outlined in the assessment. We are to create a support-vector regression model that can predict disease progression based on a set of predictor features.

Data ingestion

The diabetes dataset was imported from the scikit-learn module along with it's outcome values.

Data Cleaning

No missing values were observed and cleaning was made difficult as the dataset is already normalized so we can't really do a sense check (eg, heart beat > 0, etc). The outcome values were not normalized, but my research indicates that was not a concern for this model. Additionally, the target variable had a significant right skew, but my research also determined that log transformations were not necessary for support-vector machines.

A corrolation matrix was used to identify the degree of correlation between predictors and the target variable. I decided on a threshold of more than 0.25 or less than -0.25.

Data splititng

The data was split into testing and training sets. The training set was also broken into validation sets during the tuning phase (GridSearchCV).

Model Training

The support vector machine was modelled with the rbf kernel and then again seperately with the linear kernel on the same training set. These models were then tuned to optimize their parameters to better fit the data.

Model Selection

A naive baseline was produced using the mean value of the training sets outcome variable as discussed in the tute. A dummy regression baseline was created using the median value of predicted values using the test data. The untuned fitted model for each kernel can be used as a baseline to measure the effects of tuning parameters. The naive baseline, dummy regressor baseline, untuned models, and tuned models were used to create fitted predictions. These predictions were used in conjunction with the true values to calculate the RMSE values. The error has also been plotted with the green line indicating a model with 100% accuracy.

The two baselines had approximately the same RMSE values, likely because the dummy regressor was set to mean as well. The two tuned models had very similar RMSE values; the 'rbf' kernal producing only a slightly lower RSME value than the 'linear' kernel. The 'rbf' kernel had the best improvement after tuning, however the untuned 'linear' model performed far better than the 'rbf's untuned model. When comparing the tuned 'rbf' model to the baselines, there is clearly an improvement. The same can be said of the tuned 'linear' model. However, I would not recommend either of these models be used to predict the progression of disease. The error is far too high to make predictions regarding a persons health.
